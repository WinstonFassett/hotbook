5. Design Questions

  • Shared util vs. per-chart? → Shared util in lib/ is cleaner. Charts import as needed.
  • Cache across renders? → Yes. Cache by (text, font) key. Clear on font size changes.
  • Responsive resizing? → Font size changes need cache invalidation. Could hook into resize observer or motion config changes.
  • Truncation: ellipsis or fade? → Ellipsis is simpler and more explicit. Fade requires gradient masks.
